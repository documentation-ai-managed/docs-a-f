---
title: Create a New Project Space
description: A quick guide to starting a project in A F.
---

## Steps

<Steps>
  <Step title="Go to Dashboard">
    Navigate to `https://af-docs.com/dashboard` and log in.
  </Step>
  <Step title="Create Project">
    Click "New Project" and enter a name (e.g., "API Docs").
  </Step>
  <Step title="Add Members">
    Invite collaborators by email if needed.
  </Step>
  <Step title="Start Documenting">
    Click "Add Document," select a template, and start typing.
  </Step>
</Steps>

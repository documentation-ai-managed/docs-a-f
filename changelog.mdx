## How to Upgrade

<Callout kind="info">
Review the breaking changes in each release before upgrading your A F instance.
</Callout>

Follow these steps to upgrade to the latest version:

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your current documentation:

````bash
af-cli export --all --output ./backup-$(date +%Y%m%d)
````
  </Step>
  <Step title="Install Latest Version" icon="download">
    Use your package manager:

    <CodeGroup tabs="npm,yarn">
````bash
npm install af@latest
````
````bash
yarn add af@latest
````
    </CodeGroup>
  </Step>
  <Step title="Run Migration" icon="settings">
    Apply database migrations:

````bash
af migrate up
````
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your A F server and verify at `https://dashboard.example.com`.
  </Step>
</Steps>
